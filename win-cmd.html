Windows Important Commands

You may run several commands and scripts on your computer using the Windows command line, sometimes referred to as the Command Prompt or the Windows Terminal. The command line can offer a more effective and versatile approach to administer your system, automate operations, and solve issues—despite the fact that many users prefer to utilise the graphical user interface (GUI) on their Windows computers.

Note: The commands will be divided into a number of areas, including network setup, security, file and folder management, system information, and more. We will outline each command's syntax, parameters, and typical use situations in addition to giving usage examples. You will have a thorough understanding of the Windows command line at the end of this tutorial, as well as how to use it to improve the productivity and efficiency of your work.

Navigation Commands

# Change the current working directory to the specified path

cd [path]


# List the contents of the current directory.

dir


# create a new directory

mkdir [dir_name]


# to delete the specified directory (must be empty).

rmdir [dir_name]


# to display the contents of the specified file.

type [file_name]

File Management Commands
# copy a file to a new location.

copy [source_file] [destination]

# delete the specified file.

del [file_name]

# move a file to a new location.

move [source_file] [destination]

# to rename a file.

ren [old_file_name] [new_file_name]

System Commands
# to display system information such as the operating system name, version, and build number, as well as information about the processor, memory, and network adapters.

systeminfo

# to display a list of currently running processes. By default, it will display the process name, process ID, and memory usage for each process.

tasklist

# to terminate the specified process. The /im option specifies that the process should be terminated by its image name (i.e. the name of the executable file). For example, taskkill /im notepad.exe would terminate any instances of the Notepad application that are currently running

taskkill /im [process_name].exe

Networking Commands
# to display network configuration information such as the IP address, subnet mask, and default gateway for each network adapter. You can also use various options (e.g. ipconfig /all to display detailed information for each adapter or ipconfig /release to release the IP address for a specific adapter).

ipconfig

# to test network connectivity by sending ICMP echo request packets to the specified host.

ping [host_name]

# to trace the route that packets take from the local computer to the specified host.

tracert [host_name]

User Account Management Commands
# to manage user accounts on the local computer.

net user [username]

# to manage local groups on the local computer.

net localgroup [groupname]

Miscellaneous Commands
# to display or change file associations for a specific file extension.

assoc [.ext]

# to display environment variables such as the current user’s home directory, the system root directory, and the temporary folder location.

set

# to launch the Windows Task Manager, which displays information about currently running processes, performance metrics, and startup programs.

taskmgr

File and Directory Management Commands
# to create a new directory with the specified name.

md [directory]

# to delete a directory with the specified name.

rd [directory]

# to copy files and directories from the source location to the destination location.

xcopy [source] [destination]

System Information Commands
# Create tar named archive.tar containing directory.

systeminfo

# display detailed information about the local computer’s system configuration, including the operating system version, processor type, and installed hotfixes.

tar xf archive.tar

# display the hostname of the local computer.

hostname

# display active network connections, including the protocol, local and remote IP addresses, and connection status.

netstat

# display a list of running processes on the local computer, including the process name, process ID, and memory usage.

tasklist

# isplay a list of installed device drivers on the local computer, including the driver name, version, and file location.

driverquery

Task Scheduler Commands
# display network statistics such as active TCP connections, open ports, and listening ports.

netstat

# manage Windows services. You can use various options to start, stop, pause, resume, or query the status of a service.

sc

# Qmanage event logs on the local computer. You can use various options to view, export, or clear event logs.

wevtutil

# display the Resultant Set of Policy (RSoP) for a user or computer on the local or remote computer.

gpresult

# manage the Windows registry. You can use various options to add, delete, modify, or query registry keys and values.

reg

Batch Scripting & Processing Commands
# display messages and variables on the console.

@echo

# to create, modify, or delete environment variables.

set

# to turn off the echoing of the batch script commands on the console.

echo off

# to wait for the user to press a key before continuing the script execution.

pause

# to jump to a label in the script.

goto

# to call another batch script or command-line program from the current script.

call

# to execute a command or a series of commands based on a specific condition or a set of conditions.

if

# to execute a command or a series of commands for each item in a set of files or directories.

for

Compression and Archiving Commands
# o compress files and directories to save disk space.

compact

# to extract files from a compressed Windows cabinet (.cab) file.

expand

# to create a compressed Windows cabinet (.cab) file from one or more files.

makecab

# to create, extract, and manipulate tar archives.

tar

# to create, extract, and manipulate ZIP archives.

zip

Security Commands
# to encrypt or decrypt the files and folders on the local computer.

cipher

# to scan the system files on the local computer and repair any corrupted or missing files.

sfc

# to manage the user accounts on the local computer.

net user

# to manage the groups on the local computer.

net group

Disk Management Commands
# to check the file system and the physical disk for errors and bad sectors.

chkdsk

# to manage the disk partitions and volumes on the local computer.

diskpart

# to format the file system on a disk volume.

format

# to manage the file system features on the local computer.

fsutil

User Interface Commands
# configure the console display and input modes on the local computer.

mode

# change the foreground and background colors of the console screen on the local computer.

color

# set the title of the console window on the local computer.

title

# clear the console screen on the local computer.

cls
