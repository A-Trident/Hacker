Linux Commands

A comprehensive reference of essential Linux commands for system administration, networking, security auditing, and penetration testing. Master the command line to become an effective cybersecurity professional.

System Information
# Display Linux system information
$ uname -a
# Display kernel release information
$ uname -r
# Show which version of Red Hat is installed
$ cat /etc/redhat-release
# Show system uptime and load
$ uptime
# Show system host name
$ hostname
# Display IP addresses of the host
$ hostname -I
# Show system reboot history
$ last reboot
# Show current date and time
$ date
# Show this month's calendar
$ cal
# Display who is online
$ w
# Show current logged in user
$ whoami
Hardware Information
# Display messages in kernel ring buffer
$ dmesg
# Display CPU information
$ cat /proc/cpuinfo
# Display memory information
$ cat /proc/meminfo
# Display free and used memory (human readable)
$ free -h
# Display PCI devices
$ lspci -tv
# Display USB devices
$ lsusb -tv
# Display hardware info from BIOS
$ dmidecode
# Show info about disk sda
$ hdparm -i /dev/sda
# Perform a read speed test on disk sda
$ hdparm -tT /dev/sda
# Test for unreadable blocks on disk sda
$ badblocks -s /dev/sda
Performance Monitoring & Statistics
# Display and manage top processes
$ top
# Interactive process viewer (top alternative)
$ htop
# Display processor related statistics
$ mpstat 1
# Display virtual memory statistics
$ vmstat 1
# Display I/O statistics
$ iostat 1
# Capture and display all packets on interface eth0
$ tcpdump -i eth0
# Monitor all traffic on port 80 (HTTP)
$ tcpdump -i eth0 'port 80'
# List all open files on the system
$ lsof
# Display free and used memory (human readable)
$ free -h
# Execute "df -h" showing periodic updates
$ watch df -h
User Information & Management
# Display user and group IDs of current user
$ id
# Display the last users who logged onto the system
$ last
# Show who is logged into the system
$ who
# Create a group named "test"
# groupadd test
# Create a user account with comment and home directory
# useradd -c "John Smith" -m john
# Delete a user account
# userdel john
# Add user to a group
# usermod -aG sales john
File & Directory Commands
# List all files in detailed format
$ ls -al
# Display the present working directory
$ pwd
# Create a directory
$ mkdir directory
# Remove (delete) a file
$ rm file
# Remove directory and contents recursively
$ rm -r directory
# Force removal of file
$ rm -f file
# Copy file1 to file2
$ cp file1 file2
# Copy directory recursively
$ cp -r source_directory destination
# Rename or move a file
$ mv file1 file2
# Create symbolic link
$ ln -s /path/to/file linkname
# Create an empty file
$ touch file
# View contents of a file
$ cat file
# Browse through a text file
$ less file
# Display first 10 lines of file
$ head file
# Display last 10 lines of file and follow updates
$ tail -f file
Process Management
# Display currently running processes
$ ps
# Display all running processes on the system
$ ps -ef
# Display process information by name
$ ps -ef | grep processname
# Kill process by PID
$ kill pid
# Kill all processes by name
$ killall processname
# Start program in background
$ program &
# Bring most recent background job to foreground
$ fg
File Permissions
# Permission reference table
rwx rwx rwx = chmod 777 — Full access for all
rwx rwx r-x = chmod 775 — Full access for owner/group, read+execute for others
rwx r-x r-x = chmod 755 — Full access for owner, read+execute for group/others
rw- rw- r-- = chmod 664 — Read+write for owner/group, read for others
rw- r-- r-- = chmod 644 — Read+write for owner, read for group/others

⚠️ Use 777 sparingly!
r = Read | w = Write | x = Execute | - = No Access
Archives (TAR Files)
# Create tar archive containing directory
$ tar cf archive.tar directory
# Extract contents from archive.tar
$ tar xf archive.tar
# Create gzip compressed tar file
$ tar czf archive.tar.gz directory
# Extract gzip compressed tar file
$ tar xzf archive.tar.gz
# Create tar with bzip2 compression
$ tar cjf archive.tar.bz2 directory
# Extract bzip2 compressed tar file
$ tar xjf archive.tar.bz2
Networking
# Display all network interfaces and IP addresses
$ ifconfig -a
# Send ICMP echo request to host
$ ping host
# Display whois information for domain
$ whois domain
# Display DNS information for domain
$ dig domain
# Reverse lookup of IP address
$ dig -x IP_ADDRESS
# Download a file from URL
$ wget http://domain.com/file
# Display listening TCP and UDP ports
$ netstat -nutlp
Installing Packages
# Search for a package (RHEL/CentOS)
$ yum search keyword
# Install a package (RHEL/CentOS)
# yum install package
# Install a package (Debian/Ubuntu)
# apt install package
# Install from source code
$ tar zxvf sourcecode.tar.gz
$ cd sourcecode
$ ./configure
$ make
# make install
Search
# Search for pattern in file
$ grep pattern file
# Search recursively for pattern in directory
$ grep -r pattern directory
# Find files and directories by name
$ locate name
# Find files by name pattern
$ find /home/john -name 'prefix*'
# Find files larger than 100MB
$ find /home -size +100M
SSH Logins
# Connect to host as local username
$ ssh host
# Connect to host as specific user
$ ssh user@host
# Connect to host using specific port
$ ssh -p port user@host
File Transfers
# Secure copy file to server
$ scp file.txt server:/tmp
# Copy files from server to local
$ scp server:/var/www/*.html /tmp
# Synchronize directories with compression
$ rsync -avz /home server:/backups/
Disk Usage
# Show free and used space on mounted filesystems
$ df -h
# Display disk partitions sizes and types
# fdisk -l
# Display total disk usage of current directory
$ du -sh
Directory Navigation
# Go up one level of the directory tree
$ cd ..
# Go to the home directory
$ cd
# Change to a specific directory
$ cd /etc
