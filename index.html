What is Kali Linux?
Kali Linux — The Penetration Tester's Platform

Kali Linux is a Debian-based Linux distribution aimed at advanced Penetration Testing and Security Auditing. It contains several hundred tools geared towards various information security tasks, including Penetration Testing, Security Research, Computer Forensics, and Reverse Engineering.

Developed, funded, and maintained by Offensive Security, a leading information security training company.

  • 600+ penetration testing tools — Only the best tools included. Details on the Kali Tools site.
  • Free and always will be — Completely free of charge.
  • Open source Git tree — All source code is available for customization.
  • FHS compliant — Adheres to the Filesystem Hierarchy Standard.
  • Wide-ranging wireless device support — Compatible with numerous USB and wireless devices.
  • Custom kernel with injection patches — Latest injection patches for wireless assessments.
  • Developed in a secure environment — Committed using multiple secure protocols.
  • GPG signed packages and repositories — Every package is cryptographically signed.
  • Multi-language support — Operates in multiple languages.
Tools Required for Penetration Testing
Essential Software
  • Virtualization: VMware Workstation / Oracle VirtualBox
  • Kali Linux ISO: Download Latest
  • Vulnerable Machines: Metasploitable, DVWA, OWASP WebGoat
Recommended Hardware
  • Minimum 8 GB RAM (16 GB recommended)
  • At least 50 GB free disk space
  • Wireless adapter supporting monitor mode
  • Multi-core processor with virtualization support
